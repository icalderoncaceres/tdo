Not very well

"No muy bien" lo contrario "Very well" significa "muy bien"

Ok / Alright!

"Bien" OK tambien puede escribirse "O.K". o "okay", mientras que "Alright" tiene la variante "All right". (Estas dos expresiones también sirven para expresar conformidad).
Otras posibilidades "positivas" incluyen "Fine" y "Well"

Question words (Palabras interrogrativas)

"How...?" (¿cómo?) se útiliza delante de los sustantivos singulares.

How are you? (¿Cómo estás?)
How is she? (¿Cómo esta ella?)
How are they? (¿Cómo están ellos?)

Number (Plurals)
Número (Plurales)

Para formar los plurales en ingles mayormente basta con agregar una "S" al singular
boy   /  boys (muchachos)
eye   /  eyes (ojos)

Pero, si el singular acaba en ch, sh, s, x, z añadimos "es"

watch boy   /  watches (relojes)
box   /  boxes (cajas)

Cuando el singular acaba en "y", y la letra anterior es una consonante, la "y" se sustituye por "ies"

family   /  families (familias)
secretary   /  secretaries (secretarias)

En algunos casos, cuando termina en "f" o "fe" esta se sustituye por ves

housewife   /  housewives (amas de casa)

Plurales irregulares (algunos son idénticos a la forma singular)

man   /  men (hombres)
woman   /  women (mujeres)
person   /  people (mujeres)
sheep   /  sheep ()
chil   /  children (niños)

The time (la hora)

En ingles para decir la hora se inicia con it's, cuando la hora es exacta se aconstumbra a decir o'clock aunque no significa "en punto" como se puede creer.

01:00 / it's one o'clock.
08:00 / it's eight o'clock.
04:30 / it's half past four.
